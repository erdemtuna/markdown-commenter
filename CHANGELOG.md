### Added
- Initial release of Markdown Commenter
- VS Code extension with Copilot Chat agent integration
- CLI package for GitHub Copilot CLI users
- Interactive annotation workflow with AI-powered finding detection
- Support for Accept/Reject/Skip/Question verdicts
- GitHub-style `> [!COMMENT]` annotation format
- TypeScript annotation parsing and writing utilities
- Platform-aware agent installation (Linux, macOS, Windows)
- Resume support for partially-annotated documents
- Language Model Tool for skill retrieval in VS Code
